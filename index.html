About

ode.la is a community for creative writers. It also has an awesome writing app which acts as a muse and cures writers block. You can use the tool for free at ode.la. You can also embed the ode.la's creative writing tool on your site but Embedding that is a paid service.

Here you'll find instructions, documentation and samples to embed ode.la's writing app on your site.

This guide gives details on how to embed ode.la but it is highly recommended that you clone the repo https://github.com/ayush/ode-la-dev. Once you've finished reading this, look at the samples there and use them as a starting point.

Getting an api key

Before you proceed, you'll need an api-key from ode.la. You can contact Ayush Gupta (@ayush). Send an email describing who you are and how you wish to use this

API

odela.muse.init()

This must be called (once) to initialize odela.muse. It supports the following parameters:

odela.muse.init({
    // required
    apiKey:'[your-api-key]',  

    // optional: use this to know when odela.muse is ready
    callback: odelaReady, 

    // optional: use this to point to a different server
    service: 'http://embed.ode.la', 

    // optional: enables debug mode which loads non minified scripts and shows debug messages
    debug: true 
})

odela.muse.open()

This is called to open odela.muse app. It supports the following parameters:

odela.muse.open({
    // optional: when present populates the title
    title:'some title',  

    // optional: pre populates app with content and shows it on launch
    // if not provided, it shows blank app promoting the user to write something
    content: 'some existing content',

    // optional: for more details see below
    // when user clicks save or closes the app, it calls back this function
    callback: someFunction
})
callback param in odela.muse.open()

This is a parameter to odela.open.muse(). Its a function which received the text that a user entered after a user closes the odela muse app by clicking save or cancel/close. 

function onUserResponse(data) {
    // data.saved: true if user clicked save. false if user canceled/closed odela muse.
    // data.content: content entered by user
    // data.title: title entered by user

    // if saved flag is true then user clicked save, or else it was a cancel
    if(data.saved) {
        var userEnteredContent = data.content;
        var userEnteredTitle = data.title;
    }
}

odela.muse.close()

This can be called to force the odela.muse app to close.
