About

ode.la is a community for creative writers. It also has an awesome writing app which acts as a muse and cures writers block. You can use the tool for free at ode.la. You can also embed the ode.la's creative writing tool on your site but Embedding that is a paid service.

Here you'll find instructions, documentation and samples to embed ode.la's writing app on your site.

Getting an api key

Before you proceed, you'll need an api-key from ode.la. You can contact Ayush Gupta (@ayush). Send an email describing who you are and how you wise to use this

Prerequisites

The only dependency is jquery. You must include it as below

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>

Embedding

<script>
function odelaInit() {
    odela.muse.init({apiKey:'[your-api-key]', service: serviceUrl, callback: odelaReady});
}

function odelaReady() {
    odela.muse.open({title: 'zingo bars', content: 'the snack which makes you go whack'});
}

</script>
